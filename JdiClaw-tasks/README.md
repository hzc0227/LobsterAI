# JdiClaw 任务目录

本目录用于存放可以直接派发给 AI 的具体任务文件。

## 使用方式

1. 先阅读根目录的 `JdiClaw-execution-plan.md`
2. 从本目录选择一个任务文件
3. 新开线程时，只派发这一个任务文件
4. 要求 AI 严格遵守该任务文件里的允许范围、停止条件、完成标准和验证方式
